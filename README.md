# PineKey BMS/IIDX 输入可视化 推流插件

音乐游戏按键输入辅助显示

高亮按下的按键，统计各按键的按下次数，时长，频率，并图形化显示。
显示内容可使用OBS等直播推流应用抓取窗口。
支持键盘，手柄，游戏控制器（手台）等输入。
